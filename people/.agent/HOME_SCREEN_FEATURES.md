# Donor Home Screen - Current Features

## Overview

A compact dashboard showing the donor's impact and quick actions.

## Current Components

### 1. **App Bar**

- Donor icon with gradient background
- Greeting: "Hello, Donor!"
- Subtitle: "Make a difference today"
- Notification bell with badge indicator

### 2. **Impact Card**

Shows overall donation statistics:

- **₹25,000** - Total Donated
- **50+** - Lives Helped
- **12** - Number of Donations
- Gradient background with donor color theme
- Icons for each metric

### 3. **Ways to Help Section**

Four donation type cards in a 2x2 grid:

- **Money** (Green) - Monetary donations
- **Clothes** (Purple) - Clothing donations
- **Food** (Gold) - Food donations
- **Other** (Blue) - Other items

Each card has:

- Color-coded icon
- Type label
- Tap interaction (currently placeholder)

### 4. **Recent Activity**

Shows last 2 donations:

1. **Donated ₹5,000**

   - NGO: Hope Foundation
   - Time: 2 days ago
   - Icon: Money (Green)

2. **Clothes Donation**
   - Quantity: 25 items
   - Time: 1 week ago
   - Icon: Clothes (Purple)

Each activity item shows:

- Icon with colored background
- Title and subtitle
- Timestamp

## Visual Design

- Dark gradient background
- Glass-morphism cards
- Color-coded donation types
- Smooth spacing and padding
- Consistent border radius (20px, 16px, 12px)

## User Flow

1. User sees their impact at a glance
2. Quick access to donation types
3. Recent activity for context
4. Navigate to other tabs via bottom navigation
